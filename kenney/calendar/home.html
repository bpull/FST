Welcome to XX101!

This is the template for the home page (home.html) within the calendar 4 system. Please edit this to create a good home page for the students. Additionally you will want to edit the calendar.php file which is where all of the configuration settings for the website are located.
FilenameDescription
calendar.phpPrimary Configuration File
home.htmlHome Page
resources.htmlCourse Resources Page
policy.htmlCourse Policy Page
instructor.htmlCalendar Instructions
./calendar/Calendar Files
./calendar/cal4.virtualVirtual files (located somewhere else and automatically populated)
./calendar/cal4.accessAccess List (defines when documents will be available for students)
./calendar/calendar_display.phpDefault Calendar Layout
./calendar/calendar_display_alt.phpAlternative Calendar Layout (if desired rename this to calendar_display.php)
